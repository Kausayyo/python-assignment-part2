# Python Assignment — Part 2: Data Structures

A Restaurant Menu & Order Management System built using Python's core data structures — lists, dictionaries, and nested combinations of both.

## Tasks Covered

- **Task 1** — Explore the Menu (grouped display, stats, filtering)
- **Task 2** — Cart Operations (add, remove, update, order summary with GST)
- **Task 3** — Inventory Tracker with Deep Copy
- **Task 4** — Daily Sales Log Analysis

## File

- `part2_order_system.py` — contains all 4 tasks with comments and visible outputs

## How to Run
```bash
python part2_order_system.py
```

## Topics Used

- Nested dictionaries and lists
- `copy.deepcopy()` for safe data backup
- Loop-based filtering and sorting (no built-in `max()`/`sorted()`)
- Cart simulation with quantity management
- GST calculation and formatted order summary
- Sales log analysis across multiple days
- `enumerate` for numbered order listing
